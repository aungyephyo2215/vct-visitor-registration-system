# Visitor Registration System v1.1.0

Overview

This release completes the core visitor workflow for Condo, Apartment, Office, and Warehouse environments.

⸻

New Features

Invitation Workflow

- Create invitation
- Invitation list
- Invitation detail
- Invitation status:
  - PENDING
  - APPROVED
  - REJECTED
  - CANCELLED

⸻

Approval Workflow

- Approve invitation
- Reject invitation with reason
- Approval history
- RBAC protection

⸻

QR Workflow

- Generate QR after approval
- QR linked to Visit
- QR token validation
- QR expiration support

⸻

Badge Workflow

- Generate badge
- Print badge
- Badge linked to invitation and visit

⸻

Verification Workflow

Security guard can verify:

- Visitor name
- Phone
- ID type
- ID number
- Photo
- Vehicle number
- NDA signed
- Safety form signed

⸻

Check-in / Check-out

- QR based Check-in
- QR based Check-out
- Audit log created automatically

⸻

RBAC

Supported roles:

- SUPER_ADMIN
- PROPERTY_ADMIN
- OFFICE_STAFF
- SECURITY_GUARD
- RESIDENT
