## 🚗 Vehicle Management Module

Complete vehicle registration and management system for property security.

### Features

- **Vehicle CRUD** — Register, view, edit, and soft-delete vehicles with plate number, type, brand, color, and owner linking
- **Owner Types** — Link vehicles to residents (User) or visitors (Visitor)
- **Vehicle-Linked Visits** — Associate a registered vehicle with a visit for tracking
- **Vehicle Search** — Search by plate number, filter by owner type and status
- **Vehicle Detail Page** — View vehicle info, owner details, and visit history

### Vehicle Blacklist

- **Plate Blacklisting** — Add plates to blacklist with reason
- **Auto-Rejection** — Blacklisted plates automatically rejected at check-in
- **Blacklist Management** — Add, remove, and update blacklist entries

### Database

- New models: `Vehicle`, `VehicleBlacklist`
- New enums: `VehicleType` (8 values), `VehicleOwnerType` (RESIDENT, VISITOR), `VehicleStatus` (ACTIVE, BLOCKED)
- Visit model extended with `vehicle_id` foreign key
- 13 total migrations

## 📷 QR Camera Scanner

Browser-based camera QR scanning with complete security gate workflow.

### Features

- **Camera Scanning** — Real-time QR code detection using device camera (webcam/phone)
- **Manual Entry** — Fallback text input for QR tokens when camera unavailable
- **Camera Fallback** — Graceful degradation with clear status messages
- **Unified Token Resolver** — Resolves both QR tokens and email access tokens

### Security Gate Workflow

Complete scan-to-check-out flow on a single page:

1. **Scan** — Camera or manual QR token entry
2. **Resolve** — Token lookup (QR or email access token)
3. **Display** — Visitor summary (name, host, unit, purpose, status)
4. **Verify** — Inline verification (photo, vehicle, NDA, safety form)
5. **Check In/Out** — Status transition based on current visit state

### Key Components

- `QRScanner` — Camera-based QR scanner component
- `GateVisitorSummary` — Visit info display after scan
- `InlineVerificationDialog` — Verification form dialog
- `useSecurityWorkflow` — Gate workflow state machine hook
- `qr-token-resolver` — Unified token lookup library
