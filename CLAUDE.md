Visitor Registration System

Project Overview

Build Visitor Registration System for:

* Condo
* Apartment
* Office
* Warehouse

This system replaces paper visitor registration with QR-based digital registration.

⸻

Main Features

* Visitor Registration
* QR Code Generation
* QR Check-In
* QR Check-Out
* Visitor History
* Security Dashboard
* Audit Logs
