# Visitor Registration System

Project Overview

Build Visitor Registration System for:

* Condo
* Apartment
* Office
* Warehouse

This system replaces paper visitor registration with QR-based digital registration.

⸻

Main Features

* Pre-registration
* Walk-in Registration
* Self-Kiosk (Future)
* Visitor Information
* Visitor Type Management
* Approval Workflow
* QR Code Generation
* Badge Generation
* QR Check-In
* QR Check-Out
* Host Notification
* Visitor History
* Reports
* Security Dashboard
* Audit Logs
